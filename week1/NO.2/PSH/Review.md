[**백준 2503번 - 숫자야구**](https://www.acmicpc.net/problem/2503)

````
13* 1*2 32\* *21 *13 2\*3

3** _5_ **6

32\* *27 *37

_4_ **4 8** **8 9** _9_
```

**접근**
999가지의 경우의 수를 for문으로 다 검사해야함.
현재 검사할 값: a, 비교할 값: b
strike a와 b가 각자리에 값까지 같다면 -> strike++
a와b가 자리는 다르지만 포함된다면 -> ball++


**문제해결**
1. 1-999 번째 for뮨 (숫자가 같은 경우 제외,1-999)
   -> 아 근데 무조건 3자리니까 123부터 987까지

2. boolean이 true가 될때 count ++

3. 스트라이크를 먼저 비교: a[0] == b[0]이 같다면 -> strike ++
4. 스트라이크를 먼저 비교: a[0] == b[0]이 다르지만, b의 값이 존재한다면  -> ball++

5. 입력받은 strike와 ball값을 비교
하나라도 다르면 -> false
같으면 count ++


++ 중복제거 해야됌
++ 처음에 boolean을 배열로 만들어서 복잡해짐 -> for문 안에서 true가 될때만 count ++ 되도록

아 근데 틀렸음. 쩝.
````
