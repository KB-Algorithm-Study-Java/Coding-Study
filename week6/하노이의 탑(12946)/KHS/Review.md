**접근**
```
하노이 탑은 원반의 개수 N을 입력 받아 모든 원반을 C 막대에 옮기는 각 움직임임
한 번에 하나의 원판만 이동 가능하고 원판끼리는 크기가 더 큰 원반만 받침으로 있을 수 있음
```

**문제해결**
```
원판이 1개일때는 1 -> 3 | 1번
원판이 2개일떄는 1 -> 2, 1 -> 3, 2 -> 3 | 3번
원판이 3개일떄도 마차가지로 맨 아래있는 원판을 3번으로 보내야 하기 때문에
1. 1,2번 원판을 2번째 기둥으로
2. 3번 원판을 3번째 기둥으로
3. 1,2번 원판을 3번째 기둥으로
이 3가지 일반화된 규칙을 만족하게 된다.
```
