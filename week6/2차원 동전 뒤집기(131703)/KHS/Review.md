**접근**
```
직사각형 모양의 공간에 놓인 동전을 뒤집음
동전을 뒤집는 상태는 앞면, 뒷면 총 2가지 뿐임
각 행이나 열에 있는 동전을 뒤집을 때는 같은 줄에 있는 동전도 다 같이 뒤집힘
타겟 동전판과 똑같이 상태를 맞추는 문제
```
**문제해결**
```
접근 1.
테두리의 동전들만 뒤집어 보면 된다. 
-> 테두리의 동전들을 뒤집거만으로 내부의 동전들의 상태를 변경할 수 있어서 가능한 논리이다.

(여기서 뒤집는 동전은 각 행과 열의 첫번째 동전들이다.)
행을 기준으로 타켓 동전과 해당 행의 동전의 색이 다를 경우 뒤집고, 횟수 + 1을 진행한다.
마찬가지로 행 범위 끝까지 반복해준다.

그리고 최종적으로 열 뒤집는다.
일단 기준이 된 열 즉 0번째 열은 이미 타겟과 맞춰져 있는 상태니 건너뛴다.
그리고 이제 다음 1번째 열부터 범위 끝까지 상태 비교를 시작한다.

예를 들어 1번 열이 타켓 동전과 같은 경우
- 해당 열의 동전의 배열이 타켓 열과 같으면 다음 반복 시작
- 해당 열의 동전의 배열이 타켓 열과 다르면 뒤집고 타켓 열과 같아졌는지 확인
    여기서 같아졌으면 횟수 + 1 후 반복 계속, 다르면 다음 반복
    
이런 식으로 진행을 한다.

접근 2.
접근 1과 같은 방식으로 진행하려 하니 오류가 생겼다.
만약 열이 3개고 각 열에 동전이 000 xxx xxx 이렇게 되어 있을 때 한 번만 뒤집으면
타켓과 상태가 같아지는데 쓸데 없이 행을 기준으로 시작하니 이러한 오류가 발생

그러면 상태를 분기하자
열을 뒤집고 시작한 재귀와 안 뒤집고 시작한 재귀를 비교해서 최소 값을 결과로 출력하면 된다.
```
