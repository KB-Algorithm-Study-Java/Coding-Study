**접근**
```
n x m 격자 미로가 주어짐
출발점 (x, y) 탈출점 (r, c)이 주어짐

1. 격자 바깥으로는 나갈 수 없음
2. 이동 가능한 거리가 주어짐
3. 방문했던 격자의 재방문은 가능
4. 탈출을 시도한 문자열 중 사전순으로 가장 빠른 문자열이 탈출함
5. 왼 l, 오 r, 위 u, 아래 d - 총 4방향으로 이동 가능
```

**문제해결**
```
dfs로 구현 
맨해튼 거리 공식을 사용해 남은 거리보다 맨해튼 거리가 크거나 남은 거리 - 맨해튼 거리가 짝수가 아닌 경우로 가지치기를 하였고  
나머지는 방문배열 구현대신 스트링빌더로 문자열 넣다 뺐다를 진행
```

**후기**
> 스트링빌더 생각을 못해 방문배열 3차원을 쓰려했으나,, 구현이 어려워 스트링빌더 부분의 힌트만 받았다.
