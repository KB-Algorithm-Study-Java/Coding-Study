**접근**
```
1. 5 x 5 크기의 방이 5개가 주어짐
2. 거리두기를 위해 사람들끼리 맨해튼 거리가 2이하로 앉아야 됨
3. 하지만 맨해튼 거리가 2일때 사람 파티션 사람일 경우는 거리를 둔거로 인정
4. 각 대기실 별로 거리를 두면 1 안 두고 있으면 0을 반환
```

**문제해결**
```
1. 가독성을 위해 반마다 거리를 두었는지 확인하는 코드를 메소드로 분리하기
2. bfs를 활용해 거리를 두었는지 확인
3. 방 하나를 탐색한다 가정했을 때, 사람을 만나면 bfs 시작  
4. 맨해튼 거리 2 반경에 사람이 있는지 확인, 만약 경로 이동중 파티션을 만나면 
5. 파티션 앞에 사람이 있어도 거리를 두었다 판단하니 우회함  

```
