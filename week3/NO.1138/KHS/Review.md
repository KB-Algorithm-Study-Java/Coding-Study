https://www.acmicpc.net/problem/1138

**접근**
> 1~N의 키를 가지는 사람들이 줄을 선다.  
> 입력에는 자신 보다 키가 큰 사람이 왼쪽에 몇 명 존재하는지 주어진다.

**문제해결**
```
> 일단 입력에 사람의 총 수 N과 각 번호순 1~N까지 차례대로 왼쪽에 키 큰 사람이 몇 명 있는지 주어진다.  
> 그러면 크기가 N인 배열을 생성하고 자동으로 값이 0으로 초기화 되게 둔다.  
> 값이 0인 요소를 빈칸이라 하면 입력 받은 키 큰 사람 수는 빈칸의 수라고 생각해도 된다.  
> 따라서 빈칸의 개수를 세고 빈칸의 개수와 키 큰 사람의 수가 동일해지면 타켓 인덱스에 값을 넣는다.
```
