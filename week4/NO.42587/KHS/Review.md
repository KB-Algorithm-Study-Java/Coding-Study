**접근**
> 타겟인 프로세스는 몇 번째로 실행되는지 알아내는 문제  
> 실행 대기 큐에서 대기중인 프로세스 하나를 꺼냄  
> 남아있는 큐에 대기중인 프로세스 중 우선 순위가 더 높은 프로세스가 있다면  
> 방금 꺼낸 프로세스를 다시 큐에 넣음  
> 만약 없으면 바로 실행

**문제해결**
```
> 우선순위 큐 사용(내림차순 정렬) 
> 큐가 빌 때까지 반복하는 while문 쓰기 
> priorities 배열을 순회하며 큐의 맨 앞이랑 priorities랑 같을 때 poll(), answer++  
> 그리고 만약 loaction과 i가 같다면 return 후 종료  
```
**후기**
> 우선순위 큐를 써본 적이 없어 개념을 찾아보고 사용했다.  
> 나중에 한 번 더 풀어봐야겠다.
