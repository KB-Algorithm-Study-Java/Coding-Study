**접근**
> 각 원소가 0~9까지의 숫자로 이루어진 배열이 주어짐  
> 연속된 숫자는 중복을 제거함

**문제해결**
```
> 주어진 배열에 있는 원소를 하나씩 꺼내오면서 값을 비교  
> 이전 문자를 담을 변수 prev를 주어진 배열 arr의 인덱스 0번에 위치한 요소를 초기화  
> 초기화 진행 시 리스트에도 첫번째 요소를 담아 값의 비교는 인덱스 1번부터 시작하게 함  
> 반복문 안에 조건문을 걸어 이전 문자와 다르면 list에 add를하고 아니면 다음 반복으로 넘어가는 코드를 구현함   
```
**후기**
>  스택, 큐로 풀어야하는데 리스트로 풀어버려서 그냥 올렸습니당  
>  저번이랑 좀 다르게 생각한거는 제한사항을 보고 시간 복잡도를 좀 더 고려하면서 코드를 구현할 수 있게 된 거 같습니다.  
> O(n^2)를 넘어가면 안될 거 같아 O(n)이 걸리는 코드를 구현해야겠다 생각하고 시작했습니다.
