- 기존 main문 내부 -> int num = 10;
- 전역 -> static int num = 10; 혹은 static int num; 하고 main에서 num = 10; 으로 저장

## 🔤 문자열 (String)

문자열은 문자들의 집합으로, **문자를 하나씩 다루거나 특정 패턴을 찾는 문제**가 많이 출제된다.  
배열과 유사하지만, 문자 단위로 처리해야 한다는 점에서 차이가 있다.

- 문자열 인덱싱과 순회가 기본이다.
- 문자 비교, 변환(대소문자, 아스키 코드 등)이 자주 사용된다.
- 문자열 길이 및 부분 문자열을 활용하는 문제가 많다.

👉 핵심: **"문자를 정확하게 다루고 원하는 조건을 찾아내기"**

---

## 🛠️ 문자열 처리 (String Manipulation)

문자열 처리는 문자열을 **가공하거나 변형하는 작업**을 의미한다.

- 문자열 자르기 (`substring`)
- 문자열 나누기 (`split`)
- 문자열 합치기 (`join`, `+`)
- 특정 문자 치환 (`replace`)

또한 입력값을 문자열로 받아 처리하는 경우도 많다.

👉 핵심: **"문자열을 상황에 맞게 잘라내고, 변형하고, 재구성하기"**

---

## 🔍 문자열 응용 (String Application)

문자열을 활용한 문제는 단순 처리에서 더 나아가  
**규칙을 찾거나, 특정 패턴을 분석하는 형태**로 발전한다.

- 회문(팰린드롬) 검사
- 특정 패턴 찾기
- 문자열 압축/확장
- 빈도수 계산 (문자 개수 세기)

이 단계에서는 단순 구현 + 약간의 사고력이 함께 요구된다.

👉 핵심: **"문자열 속 규칙을 찾아 문제에 적용하기"**

---

| 문자열 응용     | 패턴 분석 + 조건에 맞는 로직 설계           |
