**접근**
```
가로 2, 세로 1인 직사각형 모양 타일이 존재
해당 타일을 이용해서 세로 3 가로 n의 바닥을 가득 채워야 함
타일 배치의 방향은 가로, 세로 다 상관 없음
주어진 바닥의 크기를 다 채울 수 있는 방법의 수 구하기
```

**문제해결**
```
일단 세로가 3이라 바닥의 가로 길이가 홀수일 경우 채울 수 있는 경우의 수는 존재하지 않음
n=2인 경우, 바닥을 채우는 경우는 3가지, 
n=4일 경우, 2일 때 경우가 두번 반복되고 이전 조합을 사용하지 않고 새로운 조합을 만드는 경우가 2번 나옴
n=6일 경우, 마찬가지로 새로 나오는 조합의 경우의 수는 2임, n=2, n=4인 경우로 나눠서 생각도 가능

1) 이전 조합을 사용해 채우는 경우
2) 이전 조합이 아닌 새로운 조합을 채우는 경우
위 2가지 경우로 구분해 점화식을 도출해야 함

일단 기본 타일을 f(2) = 3으로 설정
1. 직전 조합을 활용하는 경우 바로 앞 단계 n-2 타일 조합에 기본 타일 3가지 붙이기
f(n-2) * 3

2. 새로운 통짜 조합을 활용하는 경우
전에 타일 조합에 통짜 타일 2가지를 이어 붙임 
f(n-4) * 2 + f(n-6) * 2 + .... + f(2) * 2

점화식
f(n) = f(n-2)*3 + f(n-4)*2 + f(n-6)*2 + ..... + f(2)*2 + 2
```
