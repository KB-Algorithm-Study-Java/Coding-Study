**접근**
```
0과 1로 채워진 표에서 가장 큰 정사각형을 찾기
판 안에서 1은 1*1 정사각형 형태임
```

**문제해결**
```
현재 칸은 항상 우측 하단 꼭짓점으로 생각
정사각형이 되려면? 왼쪽, 위쪽, 왼쪽 위 대각선 칸이 1 이어야 함
현재 칸을 기준으로 정사각형이 만들어져 있어야 더 큰 정사각형을 만들 수 있음

현재 칸을 기준으로 3방향 확인 후 최솟값에 + 1을 함
최솟값을 기준으로 하는 이유는 만약 대각선만 1이면 그 쪽 방향에는 하자가 있다는 뜻이니까
```

**후기**
> 현재 값 저장을 더하는 방식으로 하는 걸 몰라서 그 부분은 힌트를 봄 DP 너무 어려웜..
